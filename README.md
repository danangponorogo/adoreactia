# Adoreactia

Boiler plate for build monolith application with AdonisJS + ReactJS + InertiaJS

# Instalation

```bash
git clone git@github.com:danangponorogo/adoreactia.git my-project
cd my-project
npm i
cp .env.example .env
```

# Hot reload

Auto reload application when we update the script. Just run:

```bash
npm run dev
```
